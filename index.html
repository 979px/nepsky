Introduction to Azure Networking

Published on February, 2023

Welcome to my Azure Networking blog, where I'll be sharing information and insights about building and managing network infrastructure in Microsoft Azure.

As more businesses move their operations to the cloud, it's becoming increasingly important to have a strong understanding of how cloud networking works. With Azure Networking, you can build and configure virtual networks, subnets, load balancers, and more to support your cloud-based applications and services. I'll be discussing the fundamentals of Azure Networking as well as more advanced topics to help you get the most out of your network infrastructure in the cloud.

Overall, having a good understanding of Azure resources is essential for building a reliable, secure, and scalable network infrastructure in the cloud.

Some topics we'll be covering include:

  • Azure Application Gateway
  • Azure AppGw Ingress Controller for AKS
  • Azure Bastion
  • Azure DDoS Protection
  • Azure DNS
  • Azure ExpressRoute
  • Azure Firewall
  • Azure Front Door and CDN
  • Azure Load Balancer
  • Azure NVA
  • Azure Peering Service
  • Azure Private Link
  • Azure Virtual Network
  • Azure Virtual WAN
  • Azure VPN

Stay tuned for more articles on Azure Networking!
